# AI Book Summarizer 📚🤖

AI Book Summarizer is a modern web application that converts long book content into concise and meaningful summaries using AI and Natural Language Processing.

Users can paste book text or upload a PDF to generate instant summaries, key insights, chapter breakdowns, and life lessons in multiple languages.

---

## 🚀 Live Demo

Try the application here:
https://ai-book-summarizer.onrender.com

---

## ✨ Features

* AI-powered book summarization
* Multiple summary styles (Concise, Detailed, Key Points, Chapter-wise)
* Multi-language summaries
* PDF upload and automatic text extraction
* Modern and responsive UI
* Custom tone options (Literary, Casual, Professional, Student)
